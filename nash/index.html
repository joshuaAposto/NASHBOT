Setting Up the App

After installing the app, follow these steps to set it up:

  • Open the NashBot app from your device.
  • On first launch, choose a model from the available options:
  • Glm4: Optimized for specific tasks.
  • Gemini: Advanced model with enhanced capabilities.
  • NASHBOT-3.5: General-purpose language model.
  • Type "help" in the chat for further instructions.
  • You can switch models at any time by typing "switch model" in the chat.

Using the Chatbot

Here’s how to interact with NashBot:

  • Type your message in the input field and press "Send" to communicate with the bot.
  • The bot's responses will depend on the selected model.
  • For specific help, type "help" in the chat.

Model Selection

To select or switch models:

  • Type "help" in the chat to see available models and instructions.
  • Choose a model by typing the model name or number.
  • For example, type "Glm4" to use the Glm4 model or "Gemini" to use the Gemini model.
