Getting Started with NashBot

Start using NashBot with these easy steps:

  • Open NashBot on your device.
  • Select a model to use:
  • Glm4: For specialized tasks and focused responses.
  • Gemini: Advanced model with comprehensive features.
  • NASHBOT-3.5: Versatile model for various needs.
  • Type "help" to get instructions and available commands.
  • Switch between models using the command "switch model".

Using NashBot

Engage with NashBot by typing your query and hitting "Send". The responses will vary based on the selected model.

For more information, type "help" to view available commands and features.

Model Selection

Change or select a model by:

  • Typing "help" to see available models and instructions.
  • Entering the model name or number (e.g., "Glm4", "Gemini") to switch.

How NashBot Can Support Students
